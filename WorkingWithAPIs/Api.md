# Importancia da API

* Ajuda trocar informações entre sistemas <br>
* Transação de dados para outros lugares/Sistemas aos quais não temos acesso <br>
* API -> Buscar e enviar dados para outros lugares <br>
* Troca de informações entre Aps e sistemas tudo padronizado <br>
* Exemplo FRONT-END COZINHA <br>
- Mesa -> Front-End = Front-end pede pro garçon <br>
- Garçon -> API = Garçon leva o pedido pra cozinha <br>
- Cozinha -> Back-End = Cozinha prepara oq o garçon pediu e devolve para o mesmo levando para o front-end novamente
